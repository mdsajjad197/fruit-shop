# Fruit Shop

A comprehensive fruit shop application with backend and frontend components.

## Features
- Real-time product browsing
- Modern and responsive UI
- Secure checkout process

## Tech Stack
- Frontend: React, Vite, Tailwind CSS
- Backend: Node.js, Express, MongoDB
- Deployment: Vercel

## Getting Started

### Prerequisites
- Node.js installed on your machine

### Installation
1. Clone the repository
   ```bash
   git clone https://github.com/mdsajjad197/fruit-shop.git
   ```
2. Install dependencies for both frontend and backend
   ```bash
   cd backend && npm install
   cd ../frontend && npm install
   ```
3. Set up environment variables in your `.env` files.
4. Run the development servers.
